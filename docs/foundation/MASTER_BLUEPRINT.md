# Purpose

The Master Blueprint defines the long-term vision, architecture principles,
engineering philosophy and development standards of the Free Basics Enterprise Platform (FBEP).

This document has the highest priority.

Every other document inside the repository must follow this blueprint.

---

# Vision

Build one of the best documented AI Commerce Platforms.

The platform must remain understandable,
maintainable,
scalable,
accessible
and secure for the next ten years.

---

# Mission

Create a modular Enterprise AI Commerce Platform that combines:

- Spreadshop
- Amazon
- CHECK24
- Tarifcheck
- Telekom

into one unified architecture.

Technology exists to serve people.

---

# Core Values

- Architecture First
- Human First
- Code Second
- Automation Third
- Quality Always
- Build Once. Scale Forever.
- Knowledge Before Code.

---

# Golden Rule

The Git repository is the Single Source of Truth.

Chats are temporary.

Documentation is permanent.

No important project knowledge may exist only inside a chat.

---

# Platform Goals

The platform shall provide:

- High-quality content
- Accessibility (WCAG 2.2 AA)
- Excellent User Experience
- Strong SEO
- AI compatibility
- High security
- Cloud scalability
- Long-term maintainability

---

# AI Strategy

The platform is AI-neutral.

Knowledge belongs to the repository.

ChatGPT, Gemini and future AI systems work from the same documented knowledge base.
